# Contribute to BlogHub Reborn

## Initial Steps

### Install Node Modules

+ Install Node Modules for Server
  ```
  cd server
  ```
  ```
  npm install
  ```

+ Install Node Modules for React App
  ```
  cd client
  ```
  ```
  npm install
  ```
  
## Running the Server and Client

+ Run the server
  ```
  cd server
  ```
  ```
  nodemon index.js
  ```
  If it errors out, type
  ```
  npm i nodemon -g
  ```

+ Run the Client
  ```
  cd client
  ```
  ```
  npm start
  ```
